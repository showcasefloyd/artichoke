# Future Enhancements

This file tracks ideas and features that are intentionally deferred beyond the current MVP phase.

## Cover Art
- Display cover images as thumbnails on issue grid cells
- Show full cover image in the issue detail modal
- Lazy-load covers to avoid unnecessary ComicVine API calls

## Local Book Rating
- Allow users to rate issues they own (e.g. 1–5 stars)
- Display aggregate rating on the series view
- Store ratings locally in the database

## Bulk Add Books
- Select multiple issues at once and mark them all as owned in one action
- Useful for onboarding a large existing collection quickly

## Bulk Delete
- Select multiple issues and remove them from the collection in one action
- Could apply to issues or entire series

## Bulk Edit
- Select multiple issues and apply a shared field update (e.g. condition, location, purchase date)
- Useful for organizing issues bought as a lot
